# SEO Инструкция для DRAWBRIDGE

## Созданные файлы

### ✅ robots.txt
- Разрешает индексацию всего сайта
- Блокирует служебные директории
- Ограничивает частоту сканирования для агрессивных ботов
- Указывает путь к sitemap

### ✅ sitemap.xml
- Карта сайта для поисковых систем
- Включает главную страницу и английскую версию
- Указаны приоритеты и частота обновлений
- **Важно**: обновляйте дату `lastmod` при изменении контента

### ✅ home-ita.html
- Английская версия страницы (нужна, чтобы `hreflang` и `sitemap.xml` не ссылались на 404)
- Должна быть доступна по URL: `https://drawbridge.kz/home-ita.html`

## Следующие шаги

### 1. Google Search Console
1. Зарегистрируйтесь на https://search.google.com/search-console
2. Добавьте сайт drawbridge.kz
3. Подтвердите владение сайтом
4. Отправьте sitemap.xml: `https://drawbridge.kz/sitemap.xml`

### Важно про hreflang
- Если вы меняете URL английской версии, обязательно синхронизируйте:
	- `link rel="alternate" hreflang=...` в [index.html](index.html)
	- записи и `xhtml:link` в [sitemap.xml](sitemap.xml)
	- сам файл английской страницы

### 2. Google Business Profile
1. Создайте профиль на https://business.google.com
2. Укажите адрес в Алматы
3. Добавьте фото офиса/команды
4. Заполните все поля (часы работы, услуги)
5. Получайте и отвечайте на отзывы

### 3. Yandex Вебмастер
1. Зарегистрируйтесь на https://webmaster.yandex.ru
2. Добавьте сайт
3. Отправьте sitemap.xml

### 4. Контент-стратегия
**Создайте блог** со статьями:
- "Как выбрать IT-подрядчика в Казахстане"
- "Интеграция с 1С: пошаговое руководство"
- "AI решения для малого бизнеса"
- Кейсы выполненных проектов
- Обзоры технологий

**Частота**: минимум 2-4 статьи в месяц

### 5. Технические улучшения
- [ ] Проверьте скорость на https://pagespeed.web.dev
- [ ] Оптимизируйте изображения (используйте WebP)
- [ ] Добавьте lazy loading для изображений
- [ ] Убедитесь, что SSL сертификат активен

### 5.1 Schema.org (структурированные данные)
- На главной добавлена разметка `FAQPage` по секции «частые вопросы» — это помогает Google понимать вопросы/ответы.
- Убран `aggregateRating` из JSON-LD, т.к. без явных отзывов/источника рейтинга на странице такой блок часто считается «самостоятельным рейтингом» и обычно игнорируется.

### 6. Обратные ссылки (Backlinks)
Разместите информацию о компании:
- **Каталоги**: 2GIS, Yandex.Maps, Google Maps
- **IT-платформы**: VC.ru, Habr, dev.to
- **Социальные сети**: LinkedIn, Facebook
- **Бизнес-каталоги**: казахстанские бизнес-справочники

### 7. Локальное SEO
- Используйте ключевые слова: "IT компания Алматы", "разработка ПО Казахстан"
- Указывайте город в контенте
- Получайте отзывы от казахстанских клиентов

### 8. Аналитика
Отслеживайте:
- Позиции по ключевым запросам
- Трафик из поисковых систем
- Конверсии (заявки с сайта)
- Поведение пользователей

### 9. Ключевые запросы для продвижения
- разработка по казахстан
- веб разработка алматы
- it компания алматы
- мобильные приложения казахстан
- интеграция 1с
- ai решения для бизнеса
- iot разработка
- devops услуги

## Ожидаемые результаты

- **1-2 месяца**: индексация сайта, первые позиции по низкочастотным запросам
- **3-4 месяца**: рост трафика, позиции в топ-30
- **6+ месяцев**: топ-10 по целевым запросам при регулярной работе

## Важно
SEO — это марафон, не спринт. Главное — регулярность и качество контента.
